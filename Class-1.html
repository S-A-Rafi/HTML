HTML Elements:

There are two types of element,

  1. Tag Element
  2. Content Element

There are three types of Tag Element,

  1. Block-level Element
  2. Inline Element
  3. Void Element

HTML Attributes:

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag2
  • Attributes usually come in name/value pairs like: name="value"

HTML Styles - CSS:

  • CSS stands for Cascading Style Sheets.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.

What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage. With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Learn more

Inline CSS

An inline CSS is used to apply a unique style to a single HTML element.

An inline CSS uses the style attribute of an HTML element.

Internal CSS

An internal CSS is used to define a style for a single HTML page.

An internal CSS is defined in the head section of an HTML page, within a style element

External CSS

An external style sheet is used to define the style for HTML pages.
